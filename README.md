# IDX-SKILLS

IDX-SKILLS is a small repository that provides a canonical `idx-search` skill and an installer script to make it available globally for Copilot, Claude, and Cursor.

## What This Repository Contains

- A canonical skill definition at `skills/idx-search/SKILL.md`
- Reference commands at `skills/idx-search/references/idx-commands.md`
- A global installer script at `install-skills.sh`
- Agent policy notes at `AGENT.md`

## Goal

The skill standardizes repository search workflows around `idx` instead of `grep`/`rg` for discovery tasks, following BM25 keyword retrieval principles.

Core behavior includes:

- Start with `idx status`
- If no index is found, ask the user before running `idx init`
- Prefer daemon-first workflow
- Use `--operator OR` and `--operator AND` when needed
- Use body match only when needed
- Run `idx sync` only when explicitly requested by the user

## Installation

From this repository root:

```bash
./install-skills.sh
```

Or choose a target directly:

```bash
./install-skills.sh copilot
./install-skills.sh claude
./install-skills.sh cursor
```

This installs the skill into the current user's home directory so it is available across projects.

## Canonical Source of Truth

The canonical skill lives in `skills/idx-search`.

All installation targets are generated by copying this canonical structure.
